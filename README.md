# Health and Fitness Tracker

## Project Overview
The Health and Fitness Tracker is designed to help users monitor and manage their fitness goals while tracking their health metrics effectively. This application will be user-friendly and offer insights into activity levels, dietary habits, and overall wellness.

## Features
- **Activity Tracking**: Log workouts, steps, and exercises to monitor physical activity.
- **Nutrition Log**: Record daily food intake to keep track of calories and nutritional information.
- **Goal Setting**: Set fitness goals and track progress over time.
- **Health Metrics**: Monitor metrics such as weight, heart rate, and calories burned.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB

## Getting Started
1. Clone the repository: `git clone https://github.com/Humbl-e/FinalYearProject.git`
2. Navigate to the project directory: `cd FinalYearProject`
3. Install dependencies: `npm install`
4. Run the application: `npm start`
